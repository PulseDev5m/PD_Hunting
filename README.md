# PD_Hunting

# #1 Fully Standalone Hunting Script

# Features
- UI/UX To See Stats And Animals Sold 
- Hunting Areas In The Config
- Sell Spots In The Config
- Database Saving So People Don't Lose Their Progress Everytime They Quit,Disconnect,Or Quit
- Animals Reset Each Time The 20 Is Hit
- Rewards At Goals Set In The UI/UX for persons liking
